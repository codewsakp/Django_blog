# Django Blogging Website

A simple blogging web application built using Django Framework.

This project was developed by following this YouTube tutorial:  
https://youtu.be/1-1ePcEDcqI?si=VyEwFusnP7rRVfUU

## Features
- User Registration and Login  
- Create, Update, Delete Blog Posts  
- Admin Panel  
- SQLite Database  
- Responsive UI  

## Technologies Used
- Python  
- Django  
- HTML, CSS, Bootstrap  
- SQLite  

## How to Run

1. Clone the repository  
git clone <repository-link>


2. Create virtual environment  
python -m venv env
env\Scripts\activate


3. Install Django
pip install django


4. Run migrations  
python manage.py migrate


5. Start server  
python manage.py runserver

6. Open in browser  
http://127.0.0.1:8000/
